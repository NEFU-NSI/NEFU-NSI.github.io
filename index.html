About Us

网络安全攻防、编程技术极客欢迎大家加入！

Network security attack and defense, programming technology geek, welcome to join!

东林网络安全与信息化协会是以网络安全和编程实战开发为主要方向的协会。协会以网络信息中心为依托，以普及网络安全知识、提高信息安全技能、加强实战开发能力为己任，为广大东林人提供一个编程极客、网络安全攻防技术交流平台。

NEFU network security and information association is an association with network security and programming development as its main direction. Relying on the network information center, the association aims to popularize network security knowledge, improve information security skills, and strengthen actual combat development ability. It provides a programming geek and network security attack and defense technology exchange platform for the majority of NEFU people
